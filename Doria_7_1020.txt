What is a disk image?
It is a copy of the data that you put on the file  after you have 
created it.
What is the command to make a disk image?
This is the command: dd if=dev/zero bs=1M count=4000 >> diskimage.img
What is a disk partition?
It is a hard disk partition that has defined storage space on a hard drive
How many disk partitions are in a disk image?
This usually depends on the copy that you want
What difficulties does this pose if any?

What steps must you take to work with this?
